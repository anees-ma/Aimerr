Technologies we use

HTML stands for Hyper Text Markup Language, which is the most widely used language on Web and is for creating web pages and web applications. With Cascading Style Sheets and JavaScript, it forms a triad of cornerstone technologies for the World Wide Web.

Adobe Photoshop is a raster graphics editor developed and published by Adobe Systems for macOS and Windows. We use Adobe Photoshop for creating images or grahical contents in order to add in web pages.

CSS stands for Cascading Style Sheets and is a style sheet language used for describing the presentation of a document written in a markup language.

jQuery is a cross-platform JavaScript library designed to simplify the client-side scripting of HTML.

JavaScript is a high-level, interpreted programming language. It is a language which is also characterized as dynamic, weakly typed, prototype-based and multi-paradigm.

Adobe Illustrator is graphic-driven software used primarily for creating vector graphics. Developed alongside with Adobe Photoshop as a companion product, Adobe illustrator is used for creating logos, graphics, cartoons and fonts for the photo-realistic layouts of Adobe Photoshop.
